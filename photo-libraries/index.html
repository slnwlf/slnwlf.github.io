A demonstration of jQuery photo zoom functionality

The first jQuery library is a classic Lightbox library. When a photo is selected, a zoomed photo open in a lightbox viewer. The lightbox can dim the background space and open with a fade in treatment.

The second jQuery library is a 'lens zoom' library. When a user mouses over a photo, a second frame opens to show the zoomed image of that area. This kind of inline zoom functionality is often seen on ecommerce sites.
